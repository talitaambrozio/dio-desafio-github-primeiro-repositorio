Estrutura de Dados: estrutura organizada na memória.

Principais Estruturas de Dados:

Vetores e Matrizes (Arrays): são estruturas de dados simples que podem auxiliar
quando há muitas variáveis do mesmo tipo em um algorítmo.

Vetor é uma estrutura de dados indexada,possui índices, que pode armazenar uma
determinada quantidade de valores do mesmo tipo.

Matriz é um vetor de vetores. É um vetor que possui duas ou mais dimensões.

Registros: é um estrutura que fornece um formato especializado para armazenar informações em memória.
Enquanto as arrays nos permitem armazenar vários dados de um único tipo, os registros nos permitem
armazenar mais de um tipo de dado.

Listas: armazenam dados de um determinado tipo em uma ordem específica.Possuem tamanho ajustável, enquanto
as arrays possuem tamanho fixo.

Lista Ligada: os nós são amarrados com indicação de qual é o próximo nó.

Lista Duplamente Ligada: são bidirecionais, sabem qual é o próximo nó e qual é o anterior.

Pilhas: estrutura de dados que serve como uma coleção de elementos e permite acesso a somente
um item de dados armazenados. Somente um item pode ser lido ou removido por vez.

Filas: admite remoção de elementos e inserção de novos sujeita à seguinte regra de operação: o elemento removido  é
o que está na estrutura a mais tempo - fifo.
